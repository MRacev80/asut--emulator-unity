## Prompt Master — скилл для генерации детальных промптов

### Ключевые идеи
- Описываешь идею в общих словах → получаешь готовый детальный промпт
- Работает с Claude Code, Codex, Cursor
- Выход: техстек, дизайн-система, архитектура, структура проекта
- По сути — **формализатор**: превращает «хочу чтоб работало» в конкретное ТЗ

### Резюме
> Скилл для агентов, который превращает расплывчатое описание в структурированную спецификацию. Полезен для генерации промптов при работе с LLM — особенно при создании эмулятора АСУТП и написании спецификаций логики ПЛК.

### Связь с моей работой
- **Эмулятор АСУТП (проект на выходные):** описать идею → получить структуру проекта
- **Спецификации логики ПЛК:** «если датчик > 200 → шнек» → детальный промпт для генерации ST
- **Скилл/агент:** можно использовать как основу для своего скилла
